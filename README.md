# 📰 Tech News — Portal de Notícias


> Portal de notícias moderno focado em tecnologia, com categorias temáticas, destaques visuais e navegação intuitiva.

---

## 📌 Sobre o Projeto

O **Tech News** é um portal de notícias desenvolvido para agregar e exibir as principais informações do mundo da tecnologia. Com um layout clean e responsivo, o usuário pode navegar por diferentes categorias, ler destaques da semana e acompanhar notícias em tempo real.

## ✨ Funcionalidades

- 🗂️ **Categorias temáticas** — Inteligência Artificial, Blockchain, Hologramas, Internet, Vestíveis, Realidade Aumentada, Realidade Virtual e mais
- 🔥 **Destaques da semana** — Cards com as notícias mais relevantes em destaque
- 🔍 **Campo de busca** — Pesquisa rápida por notícias
- 🏷️ **Tags por categoria** — Identificação visual do tema de cada notícia
- 📱 **Layout responsivo** — Adaptado para desktop e mobile
- 📰 **Feed de notícias** — Seção "Mais lidas" e destaques por categoria

## 🚀 Como Executar

### Pré-requisitos

Nenhuma dependência externa é necessária. Basta ter um navegador moderno instalado.

### Passos

1. **Clone o repositório:**
   ```bash
   https://github.com/leonardoSilva0022/fullstack-projeto-portal-de-noticias.git
   ```

2. **Acesse a pasta do projeto:**
   ```bash
   cd fullstack-projeto-portal-de-noticias
   ```

3. **Abra o arquivo principal no navegador:**
   ```bash
   # Opção 1: abrir diretamente
   open index.html

   # Opção 2: usar extensão Live Server no VS Code
   # Clique com o botão direito em index.html > "Open with Live Server"
   ```
